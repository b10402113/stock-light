三、各組件職責詳解

1. Ingress / Nginx

職責：
├── 反向代理
│ ├── / → React 前端
│ ├── /api/\* → FastAPI 後端
│ └── /webhook/line → FastAPI (LINE webhook)
│
├── SSL/TLS 終止
│ └── HTTPS → HTTP (卸載加密)
│
├── 負載均衡
│ └── 請求分發到多個 FastAPI Pod
│
└── 靜態資源緩存
└── React JS/CSS 緩存

2. React 前端

職責：
├── 提供靜態資源
│ ├── index.html
│ ├── main.js
│ └── styles.css
│
└── 用戶介面
├── 訂閱管理頁面
├── 儀表板
└── 用戶設定

3. FastAPI 後端

職責：
├── API 請求處理
│ ├── 用戶註冊/綁定
│ ├── 訂閱 CRUD
│ ├── 股票查詢
│ └── 指標計算
│
├── LINE Webhook 處理
│ ├── 接收 LINE 訊息
│ ├── 解析指令
│ └── 觸發異步任務
│
├── 業務邏輯
│ ├── 權限驗證
│ ├── 額度檢查
│ └── 數據驗證
│
└── 任務調度
└── 發送任務到 Celery (via Redis)

4. Postgres

職責：
├── 數據持久化
│ ├── users (用戶表)
│ ├── stocks (股票表)
│ ├── stock_prices (價格歷史)
│ ├── subscriptions (訂閱表)
│ └── notification_logs (通知記錄)
│
└── 事務支持
└── ACID 保證

5. Redis

職責：
├── Celery Broker
│ └── 任務隊列
│
├── 快取層
│ ├── 股票價格快取 (TTL: 5min)
│ ├── API 回應快取 (TTL: 1min)
│ └── 用戶 Session
│
└── 速率限制
└── API 調用計數

6. Celery Worker

職責：
├── 異步任務執行
│ ├── update_stock_prices (更新股價)
│ ├── calculate_indicators (計算指標)
│ ├── check_alert_conditions (檢查觸發)
│ └── send_notifications (發送通知)
│
└── 外部 API 調用
├── 富果 API
├── LINE Messaging API
└── OpenAI API

7. Celery Beat

職責：
├── 定時任務調度
│ ├── 09:05 - 開盤初始化
│ ├── \*/5min - 檢查價格觸發 (交易時間)
│ ├── 14:30 - 收盤更新價格
│ ├── 15:00 - 計算日線指標
│ └── Daily - 清理過期數據
│
└── 任務發送到 Redis Queue
